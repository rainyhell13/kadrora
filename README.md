# Kadrora — Социальная сеть

Дипломный проект: полноценная социальная сеть на PHP + PostgreSQL.

---

## Технологии

| Уровень     | Технология                        |
|-------------|-----------------------------------|
| Backend     | PHP 8.1+ (чистый MVC, без фреймворка) |
| База данных | PostgreSQL 14+                    |
| Frontend    | Bootstrap 5.3, Vanilla JS         |
| Иконки      | Bootstrap Icons 1.11              |
| Веб-сервер  | Встроенный PHP-сервер (`php -S`)  |

---

## Быстрый старт

### Требования

- **PHP 8.1+** (проверить: `php --version`)
- **PostgreSQL 14+** запущен локально (порт 5432, пользователь `postgres`, пароль `1`)

### Запуск

Из терминала, находясь в папке проекта:

```bash
cd social_network
php -S 127.0.0.1:8000 router.php
```

Открыть в браузере: **http://127.0.0.1:8000**

> На Windows используйте именно `127.0.0.1`, а не `localhost` — иначе возможна ошибка
> `ERR_CONNECTION_REFUSED` из-за несовпадения IPv4/IPv6. Приложение само подстраивает
> адреса ссылок под тот хост, по которому вы открыли страницу.

> **База данных создаётся автоматически** при первом запуске — никаких действий в pgAdmin не нужно. Схема, таблицы и тестовые данные применяются сами.

## Функционал

- **Регистрация / Вход** — валидация, CSRF-защита, хеширование паролей (bcrypt)
- **Тёмная / светлая тема** — переключение в шапке, сохраняется между сессиями
- **Левое меню** — навигация по разделам на всех страницах
- **Профиль** — аватар, обложка, статус, стена, подробная анкета («Показать подробную информацию»)
- **Стена** — записи на своей странице и на стене друзей
- **Лента новостей** — посты друзей и свои посты, пагинация
- **Посты** — текст + фото, настройка приватности (все / друзья / только я)
- **Лайки и комментарии** — без перезагрузки страницы (AJAX)
- **Закладки** — сохранение понравившихся записей
- **Друзья** — заявки, подтверждение, отклонение, удаление, общие друзья
- **Сообщества (группы)** — создание, управление, записи, участники
- **Личные сообщения** — чат с автообновлением (polling каждые 3 сек)
- **Фотоальбом** — загрузка, просмотр, удаление фото
- **Видеозаписи** — загрузка и просмотр видео
- **Документы** — загрузка, хранение и скачивание файлов
- **Уведомления** — лайки, комментарии, заявки, записи на стене
- **Поиск** — по имени, фамилии, никнейму, городу
- **Онлайн-статус** — отображается в профиле и в чате

## База данных

### Таблицы

| Таблица                   | Назначение                              |
|---------------------------|-----------------------------------------|
| users                     | Пользователи                            |
| posts                     | Посты                                   |
| post_likes                | Лайки постов                            |
| comments                  | Комментарии (с вложенностью)            |
| comment_likes             | Лайки комментариев                      |
| friendships               | Дружба / заявки / блокировки            |
| conversations             | Диалоги мессенджера                     |
| conversation_participants | Участники диалогов                      |
| messages                  | Сообщения                               |
| notifications             | Уведомления                             |
| photos                    | Фотоальбом                              |

### Особенности

- Триггер автоматически обновляет `likes_count` в таблице `posts`
- Триггер обновляет `updated_at` при каждом изменении записи
- Полнотекстовый поиск через `tsvector` (индекс GIN) по пользователям
- Индексы на все часто используемые поля
- База и схема создаются автоматически при первом запуске

## Настройка подключения к БД

Файл `config/database.php`:

```php
define('DB_HOST',     'localhost');
define('DB_PORT',     '5432');
define('DB_NAME',     'kadrora');
define('DB_USER',     'postgres');
define('DB_PASSWORD', '1');
```

Если нужно сбросить БД и пересоздать заново — удалите файл `database/.initialized`.
